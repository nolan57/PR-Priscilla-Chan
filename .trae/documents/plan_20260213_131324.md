## 为界面添加暗黑模式

### 实现方案

#### 1. 创建主题样式字典
在 `ProSingerSeparatorApp` 类中添加亮色/暗色主题样式定义：

```python
THEMES = {
    'light': {
        'bg_primary': '#ffffff',
        'bg_secondary': '#fafafa',
        'bg_group': '#f5f5f5',
        'text_primary': '#2c3e50',
        'text_secondary': '#555555',
        'border': '#bdc3c7',
        'border_hover': '#3498db',
        'accent': '#3498db',
        'success': '#27ae60',
        'success_bg': '#e8f5e9',
        'waveform_bg': '#ffffff',
    },
    'dark': {
        'bg_primary': '#1e1e1e',
        'bg_secondary': '#2d2d2d',
        'bg_group': '#252526',
        'text_primary': '#e0e0e0',
        'text_secondary': '#b0b0b0',
        'border': '#404040',
        'border_hover': '#4fc3f7',
        'accent': '#4fc3f7',
        'success': '#4caf50',
        'success_bg': '#1b5e20',
        'waveform_bg': '#1a1a1a',
    }
}
```

#### 2. 添加主题切换按钮
在界面顶部添加主题切换按钮

#### 3. 创建 `apply_theme` 方法
统一应用主题到所有组件，包括：
- 主窗口背景
- QGroupBox 分组框
- QFrame 框架
- QLabel 标签
- QComboBox 下拉框
- QPushButton 按钮
- QScrollArea 滚动区域
- QStatusBar 状态栏
- WaveformDisplayWidget 波形显示
